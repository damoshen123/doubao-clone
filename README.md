# 豆包声音复刻工具

一键部署你自己的声音复刻服务。

## 部署方法

点击下方按钮，一键部署到 Vercel：

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/你的用户名/doubao-clone)

## 使用说明

1. 在 [火山引擎控制台](https://console.volcengine.com/speech/service/8) 获取 Access Token 和 APP ID
2. 创建 Speaker ID
3. 上传音频进行声音复刻

## 安全说明

- 您的 Token 仅保存在浏览器本地
- 服务器不存储任何用户数据
